# Exercise 0.26 — Mini Library

## Scenario

You are building a small library system. Books are stored in a CSV file.

## What to Build

A menu-driven program with three features:

```
1. Add book
2. List all books
3. Search by title
4. Quit
```

Each book has: title, author, year.

## What to Observe

As you add features, notice what happens to the code. Does it stay easy to read? Easy to change?

This is the first time you will feel the code becoming hard to manage.

## Transition

This is the last exercise of Part 0. The project is growing. That growth is the reason Part 1 exists.
