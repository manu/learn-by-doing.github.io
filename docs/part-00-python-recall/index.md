# Python Recall Bootcamp

This is not a Python course. It is a Python *recall* course.

You already know the basics from Class 10. This section brings that knowledge back to life through real problems — each one introducing exactly one new idea, building up to the Mini Library that kicks off the whole project.

## Rules for this section

- Write short programs — 20 to 50 lines at most
- No classes, no frameworks
- Focus on clarity and getting things working

## The Arc

Each exercise introduces one new concept. By the end you will have used all the building blocks needed for the main project.
