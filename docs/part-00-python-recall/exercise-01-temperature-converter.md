# Exercise 0.1 — Temperature Converter

## Scenario

Weather data arrives in Fahrenheit. Local reports need Celsius.

## What to Build

A program that asks the user for a temperature in Fahrenheit and prints it in Celsius.

```
Enter temperature in Fahrenheit: 98.6
That is 37.0 degrees Celsius.
```

## What to Observe

- How do you get input from the user?
- How do you store a number in a variable?
- How do you apply a formula?

## The Formula

```
celsius = (fahrenheit - 32) × 5 / 9
```

## Concept

Python syntax recall — variables, input, arithmetic, print.
