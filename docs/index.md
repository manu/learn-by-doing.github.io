# Engineering Through One Project

A 2-year engineering curriculum for Amal, built around one evolving project: a **Library Management System**.

The goal is not to memorize syntax. The goal is to understand *why* each technology exists.

> "What problem was this technology invented to solve?"

---

## The Project

Every part of this curriculum adds to the same library system — starting as a tiny Python script, and growing into a deployed web application with a real database, security, networking, and automation.
