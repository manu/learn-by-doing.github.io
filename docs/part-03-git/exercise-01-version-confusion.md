# Exercise 3.1 — Version Confusion

## Scenario

You open a project folder and find these files:

```
library.py
library_final.py
library_final2.py
library_final_FIXED.py
library_final_latest.py
```

## What to Do

1. Open each file and try to figure out which one is the "real" version
2. Write down what was confusing about the process

## What to Observe

- How long did it take to find the right version?
- What information was missing?
- What would have made this easier?

## The Question

What problem would be solved if the computer tracked every change, with a message explaining why?

## Technology Introduced Next

Git — a tool that keeps a complete history of every change, with the author and reason recorded.
