# Git & History

Before learning Git commands, you will feel the pain that Git was invented to solve.

You will look at a folder with files named `library.py`, `library_final.py`, `library_final2.py`, and `library_final_latest.py` — and try to figure out which one is correct. That frustration is the reason Git exists.

## Key Concepts

- History and commits
- Branches
- Issues
- Pull Requests

## The Rule

Learn the *meaning* before the command.
