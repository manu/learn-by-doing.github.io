# Engineering Mindset

Code is written once. It is read many times.

This section is about how engineers think before they write code — how they name things, organize files, and communicate intent to other people (and to their future selves).

## Key Concepts

- File and folder organization
- Naming variables and functions
- Writing a README
- Information architecture
