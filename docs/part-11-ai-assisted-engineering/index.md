# AI-Assisted Engineering

AI can generate Flask code, SQL queries, Git commands, and Tailwind layouts. That is not the skill. The skill is knowing whether what it generated is correct, secure, and well-designed.

## What AI Can Generate

- Flask routes and views
- SQL queries and migrations
- Git commit messages
- Tailwind CSS layouts
- Test cases

## What Engineers Must Still Do

- Understand the architecture
- Identify security risks
- Design the data model
- Make trade-off decisions
- Review and refactor AI output

## The Question

If AI wrote this code, how would you know it is correct?
